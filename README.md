# 🇪🇸 西班牙语学习 / Spanish Learning

基于 40 篇西语短文的交互式学习网页，支持逐句精析、语法笔记、TTS 语音朗读。

## 功能

- 📖 40 篇西班牙语短文（西语原文 + 中文翻译）
- 📝 逐句语法/发音/词性笔记（预填 + 可编辑）
- 🔊 双引擎语音朗读（Google TTS + Web Speech API）
- 🎨 响应式设计（桌面 + 手机端）
- 💾 笔记自动保存至 localStorage
- 🖨️ 打印导出优化

## 使用

直接用浏览器打开 `index.html`，或部署到 GitHub Pages。

### 本地运行

```bash
open index.html
# 或启动本地服务器以启用 Google TTS：
python3 -m http.server 8080
```

### 部署

推送后在 GitHub Settings → Pages 启用，选择 main 分支根目录即可。
